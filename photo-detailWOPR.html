WOPR

“A strange game. The only winning move is not to play.”
(WOPR, WarGames)

Straight from the 1983 film WarGames this 3D printed model of the WOPR (War Operation Plan Response) features randomly alternating red and yellow NeoPixels as well as a button activated audio clips. All the programming, painting, and wiring for this project was completed by me with the CAD work completed by Thingiverse user Corey. The biggest challenge with this project was fitting all the electronics in such a small profile. The STLs can be found here: WOPR
