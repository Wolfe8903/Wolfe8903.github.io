Mechanical Iron Man Mk. 3 Helmet

“I don’t want to see this on your myspace page. Please no gang signs.”
(Tony Stark, Iron Man)

The Mechanical Iron Man Mk. 3 Helmet was fully printed, wired and assembled by myself. The CAD work was completed by Cults user BOXANDLOOP. This helmet features 12 servos along with blue LED eyes for the classic Iron Man look. The biggest challenge with this project came from dealing with the servos. Having very little experience with arduino and C++, it was challenging to learn how to read and understand all the intricacies of the provided code. In addition due to the large number of servos an external PWM Servo Driver was neccesary for this project. The headers for this driver did not come pre-soldered so I was required to learn how to solder with a level of precision not needed previously. As an additional note this project taught me that not all components are created equal as many months of work on this project were wasted solely due to me attempting to use an pre-soldered PWM Driver from Amazon as opposed to the recommended driver from Adafruit.
